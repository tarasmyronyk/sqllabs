# База даних "Організація автобусних екскурсій" 

Проєкт створено в рамках лабораторних робіт з курсу баз даних та знань (СУБД PostgreSQL). 

Варіант 4: Екскурсії.

Структура проєкту
У цьому репозиторії знаходяться SQL-скрипти для створення, наповнення та управління базою даних автобусного парку.

* `sqllab1bd.sql` — створення таблиць (Екскурсії, Автобуси, Замовники тощо), налаштування зв'язків (Foreign Keys) та обмежень цілісності.
* `lab2queries.sql` — SQL-запити для вибірки даних (JOIN, GROUP BY, підзапити) та генерації бізнес-звітів (графік використання автобусів, фінансовий звіт за місяць).
* `lab3procedures.sql` — PL/pgSQL процедури для автоматичного розрахунку та нарахування оплати клієнтам за вказаний період.
